# PyQt5 Projects

This repository contains two PyQt5-based desktop applications.

## Projects Included

### ⏰ Digital Clock
- Displays real-time digital clock
- Built using PyQt5 and QTimer

### ⏱ Stopwatch
- Start, stop, and reset functionality
- Built using PyQt5 GUI components

## Tech Stack
- Python
- PyQt5

## How to Run
```bash
pip install PyQt5
python digital_clock.py
python stopwatch.py
